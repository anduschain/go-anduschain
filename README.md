## Go Anduschain

Official golang implementation of the andusdeb protocol.

## System requirements

```
it is recommended to have at least 4 GB of memory available when running anduschain.
OS : Linux(CentOS, Ubuntu), Mac, Windows
CPU : least 4 CORE CPU
```
